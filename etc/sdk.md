# SDKMAN 활용
## Description
SDKMAN을 이용하여 자바 버전을 관리하는 법.

## 순서
1. 자바 버전 확인하기
~~~bash
sdk list java
~~~
2. 확인한 버전을 토대로 자바 설치하기
~~~bash
sdk install java [version]
~~~
3. 설치 된 자바를 기본으로 설정하기
~~~bash
sdk default java [version]
~~~
4. JRE 버전 확인하기
~~~bash
java --version
~~~
5. Compiler 버전 확인하기
~~~bash
javac --version
~~~
