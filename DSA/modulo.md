# modulo 연산 - 덧셈법칙

## 공식
> (a + b) % m = (a % m + b % m) % m

나누는 수 m 이 존재 할 때, 두 수의 합의 나머지와 각각의 수의 나머지를 합한 것의 나머지는 동일하다.

### 중심요소
- 공식 자체
- (a + b) = (a) + (b) -> X = x1 + x2의 관계일 때 활용 


## 활용
한 배열에서 특정 구간들이 특정한 수로 나눠질 경우 해당 구간들의 개수를 셀 때 활용할 수 있다.

### 적용 방법
1. 좌항 적용
2. 우항 적용
3. 좌항과 우항의 값 비교
